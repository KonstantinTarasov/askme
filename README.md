# Хорошие вопросы

Cоциальная сеть, предназначеная для задания вопросов пользователю и возможностью ответа на них.

### Версия Ruby

* 2.7.0

### Инструкция по развертыванию

* bundle
* yarn install
* rake db:migrate
* rails s
* 0.0.0.0:3000

### Приложение на Heroku

[Хорошие вопросы](https://super-askme.herokuapp.com/)
